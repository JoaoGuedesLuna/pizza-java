<h3>🛠 Funcionalidades</h3>
<ul>
  <li>
    <b>Cadastro:</b> permite aos usuários criar uma conta na aplicação fornecendo informações pessoais.
  </li>
  <br>
  <li>
    <b>Login:</b> permite que os usuários acessem suas contas existentes na aplicação. Ele inserem suas credenciais para acessar as funcionalidades da aplicação.
  </li>
  <br>
  <li>
    <b>Visualização de produtos:</b> permite aos usuários navegar e visualizar os produtos disponíveis no menu da pizzaria.
  </li>
  <br>
  <li>
    <b>Carrinho:</b> o carrinho é onde os usuários podem adicionar produtos que desejam comprar. Eles podem adicionar, remover itens e definir quantidades.
  </li>
  <br>
  <li>
    <b>Compra:</b> a funcionalidade de compra permite aos usuários finalizar seus pedidos, selecionar a forma de pagamento, fornecer informações de entrega e confirmar a compra. Após a compra, os usuários geralmente recebem uma confirmação do pedido e um tempo estimado de entrega.
  </li>
  <br>
  <li>
    <b>Histórico de pedidos:</b> o histórico de pedidos é uma seção que permite aos usuários ver um registro de todas as compras anteriores que fizeram na aplicação.
  </li>
  <br>
  <li>
    <b>Atualização de dados:</b> essa funcionalidade permite aos usuários atualizar as informações de sua conta, como endereço de entrega, número de telefone, senha e outros detalhes pessoais. 
  </li>
</ul>

<h3>👨‍💻 Tecnologias usadas</h3>
<ul>
  <li>
    ☕ Linguagem <b>Java</b> (JDK 17) tanto para desenvolvimento Back-End como para desenvolvimento Front-End (a IDE usada para o Front-End foi o Netbeans).

  <br>
  <li>

<h3>▶ Executando a Aplicação</h3>
<h4>.jar</h4>
<p>
  Uma forma de executar o programa é por meio do arquivo . jar que se encontra na diretório /dist.
</p>
<h4>IDE</h4>
<p>
  É possível executar o programa por meio de uma IDE, nesse caso deve-se adicionar as bibliotecas que se encontram no diretório /lib à estrutura do projeto na IDE.
</p>
