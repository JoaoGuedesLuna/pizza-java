### Versão 1.0 

**💻 Tecnologias usadas:** 

* ☕ Linguagem Java (JDK 17) tanto para desenvolvimento Back-End como para desenvolvimento Front-End (a IDE usada para o Front-End foi o Netbeans). 

* 🎲 Banco de Dados H2 em memória para o armazenamento de dados.

  

**⚙ Padrões de Projeto:**

* Padrão de projeto DAO para persistir os dados no banco de dados.



### Executando a Aplicação ▶



### .jar

Uma forma de executar o programa é por meio do arquivo . jar que se encontra na diretório /dist.



### IDE

É possível executar o programa por meio de uma IDE, nesse caso deve-se adicionar as bibliotecas que se encontram no diretório /lib à estrutura do projeto na IDE.
